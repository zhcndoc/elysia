Eden 测试 ​

使用 Eden，我们可以创建一个具有端到端类型安全和自动补全的集成测试。

使用 Eden Treaty 创建测试，由 irvilerodrigues 在 Twitter 上

设置 ​

我们可以使用 Bun test 来创建测试。

在项目目录的根部创建 test/index.test.ts，内容如下：

typescript
// test/index.test.ts
import { describe, expect, it } from 'bun:test'

import { edenTreaty } from '@elysiajs/eden'

const app = new Elysia()
    .get('/', () => 'hi')
    .listen(3000)

const api = edenTreaty<typeof app>('http://localhost:3000')

describe('Elysia', () => {
    it('返回响应', async () => {
        const { data } = await api.get()

        expect(data).toBe('hi')
    })
})

然后，我们可以通过运行 bun test 来执行测试。

bash
bun test

这使我们能够以编程方式执行集成测试，而不是手动获取，同时自动支持类型检查。
